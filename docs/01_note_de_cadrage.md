# Note de cadrage — Outil.Nutri

## Vision
Outil.Nutri est un projet de plan de ravitaillement multi-sport destiné aux athlètes d’endurance.

## Problème à résoudre
Les athlètes préparent souvent leur nutrition de manière approximative, générique ou peu adaptée à leur sport, à leur profil digestif et à la logistique réelle de leur course.

## Cible prioritaire
Athlètes d’endurance pratiquant le trail, la course sur route, le cyclisme et le triathlon, avec une priorité de lancement sur les efforts longs où le risque d’erreur nutritionnelle est élevé.

## Sports couverts
- Trail
- Course sur route
- Cyclisme
- Triathlon

## Proposition de valeur
Générer un plan de ravitaillement personnalisé selon le sport, la durée, l’intensité, le profil utilisateur, les contenants disponibles et, plus tard, le parcours réel.

## Périmètre MVP
- profil utilisateur
- création d’une activité
- moteur de calcul
- plan avant / pendant / après
- répartition par contenant
- historique simple
- feedback post-effort

## Hors périmètre
- live
- Garmin / Wahoo
- machine learning
- GPX avancé
- marketplace produits

## Différenciation
Le projet vise à dépasser les calculateurs statiques en combinant logique métier, contraintes digestives, logistique réelle et future contextualisation terrain.

## Critères de succès
- plan compréhensible
- plan physiologiquement cohérent
- utilité perçue sur le terrain
- possibilité d’ajustement après feedback utilisateur
