# Proposition de valeur — Outil.Nutri

## Version courte
Outil.Nutri génère un plan de ravitaillement personnalisé pour le trail, la course sur route, le cyclisme et le triathlon, selon le sport, la durée, l’intensité, le profil utilisateur et les contenants disponibles.

## Version claire
Les sportifs d’endurance préparent souvent leur nutrition de manière approximative, trop générique ou difficile à exécuter sur le terrain.

Outil.Nutri vise à résoudre ce problème en transformant des données simples :
- sport
- durée
- intensité
- température
- poids
- tolérance digestive
- logistique disponible

en un plan concret :
- quoi prendre
- combien prendre
- sous quelle forme
- quand le prendre
- comment le répartir dans les contenants

## Problème principal
Aujourd’hui, beaucoup d’athlètes :
- improvisent leur ravitaillement
- sous-estiment l’hydratation ou le sodium
- se fient à des conseils trop généraux
- ne tiennent pas compte de leur propre tolérance digestive
- n’ont pas de plan réellement exécutable pendant l’effort

## Promesse produit
Passer d’un ravitaillement approximatif à un plan structuré, compréhensible et applicable.

## Différenciation
Outil.Nutri ne se limite pas à donner une quantité théorique par heure.

Le projet vise à combiner :
- logique métier par sport
- prudence digestive
- contraintes réelles de transport
- adaptation progressive au profil utilisateur
- plus tard, contextualisation au parcours réel

## Pour qui
- trailers
- coureurs sur route
- cyclistes
- triathlètes

avec une priorité sur les efforts d’endurance où le risque d’erreur nutritionnelle est élevé.

## Bénéfices utilisateur
- gagner en clarté
- éviter les erreurs grossières
- mieux structurer son effort
- réduire le risque de coup de mou
- réduire le risque digestif
- mieux préparer ses contenants avant une séance ou une course

## Positionnement MVP
Le MVP n’est pas un coach nutritionnel complet.
Le MVP est un générateur de plan de ravitaillement multi-sport, simple, prudent et utile.

## Formule de synthèse
Outil.Nutri transforme les variables clés d’un effort d’endurance en un plan de ravitaillement concret, lisible et cohérent.
